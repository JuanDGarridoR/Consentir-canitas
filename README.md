# 🧓📲 Consentir Canitas – Sistema de Asistencia con QR

Sistema de información orientado a la **gestión y análisis de datos de adultos mayores**, desarrollado como parte de un **Proyecto Social Universitario** de la Pontificia Universidad Javeriana.

---

## 📌 Descripción

**Consentir Canitas** es una aplicación que permite registrar la asistencia de adultos mayores mediante el uso de **códigos QR**, facilitando la organización, el seguimiento y la toma de decisiones dentro de comunidades con recursos limitados.

El sistema está diseñado específicamente para contextos con:
- Baja conectividad  
- Acceso limitado a tecnología  
- Necesidad de soluciones simples y funcionales  

---

## 🎯 Objetivo

Optimizar la gestión de información del grupo comunitario mediante una herramienta tecnológica que permita:

- Registrar asistencia de manera rápida  
- Almacenar datos básicos de los participantes  
- Generar reportes y análisis de participación  
- Reducir riesgos en situaciones de emergencia  

---

## 🚨 Problemática

Actualmente, la comunidad presenta las siguientes dificultades:

- Registro de asistencia manual en cuadernos  
- Ausencia de datos médicos y contactos de emergencia  
- Falta de análisis de información  
- Limitaciones tecnológicas  

---

## 💡 Solución

Se propone una aplicación móvil con las siguientes funcionalidades:

### 📱 Funcionalidades principales

- Escaneo de códigos QR para registrar asistencia  
- Registro de adultos mayores  
- Generación de carnets personalizados con QR  
- Almacenamiento de datos básicos (nombre, edad, contacto, etc.)  
- Historial de asistencia  
- Generación de reportes (diarios, semanales, mensuales)  

---

## 🛠️ Tecnologías (propuestas)

- Aplicación móvil (Android)  
- Generación de códigos QR  
- Base de datos (local)  
- Exportación a Excel / Google Sheets  
- Backend ligero

## 🏘️ Contexto

El sistema está dirigido al grupo comunitario **Consentir Canitas**, ubicado en Usme (Bogotá), donde:

- Asisten entre 50 y 60 adultos mayores semanalmente  
- No existe infraestructura tecnológica  
- Se desarrollan actividades recreativas y de bienestar  

---

## 📊 Impacto Esperado

- Mejor organización interna  
- Digitalización de procesos  
- Toma de decisiones basada en datos  
- Reducción de riesgos en emergencias  
- Mayor facilidad para gestionar apoyos institucionales  

---

## ⚠️ Consideraciones

- El sistema debe ser simple y fácil de usar  
- Debe funcionar con conectividad limitada  
- Requiere capacitación básica para los usuarios  
- Debe garantizar la privacidad de los datos

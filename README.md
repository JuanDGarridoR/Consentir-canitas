# 🧓📲 Consentir Canitas – Sistema de Asistencia con QR

Sistema de información orientado a la **gestión y análisis de datos de adultos mayores**, desarrollado como parte de un **Proyecto Social Universitario** de la Pontificia Universidad Javeriana.

---

## 📌 Descripción

**Consentir Canitas** es una aplicación que permite registrar la asistencia de adultos mayores mediante el uso de **códigos QR**, facilitando la organización, el seguimiento y la toma de decisiones dentro de comunidades con recursos limitados.

El sistema está diseñado específicamente para contextos con:
- Baja conectividad  
- Acceso limitado a tecnología  
- Necesidad de soluciones simples y funcionales  

---

## 🎯 Objetivo

Optimizar la gestión de información del grupo comunitario mediante una herramienta tecnológica que permita:

- Registrar asistencia de manera rápida  
- Almacenar datos básicos de los participantes  
- Generar reportes y análisis de participación  
- Reducir riesgos en situaciones de emergencia  

---

## 🚨 Problemática

Actualmente, la comunidad presenta las siguientes dificultades:

- Registro de asistencia manual en cuadernos  
- Ausencia de datos médicos y contactos de emergencia  
- Falta de análisis de información  
- Limitaciones tecnológicas  

---

## 💡 Solución

Se propone una aplicación móvil con las siguientes funcionalidades:

### 📱 Funcionalidades principales

- Escaneo de códigos QR para registrar asistencia  
- Registro de adultos mayores  
- Generación de carnets personalizados con QR  
- Almacenamiento de datos básicos (nombre, edad, contacto, etc.)  
- Historial de asistencia  
- Generación de reportes (diarios, semanales, mensuales)  

---

## 🛠️ Tecnologías (propuestas)

- Aplicación móvil (Android)  
- Generación de códigos QR  
- Base de datos (local)  
- Exportación a Excel / Google Sheets  
- Backend ligero

# ⚙️ Guía de Instalación en Dispositivo Android

## 📋 Requisitos mínimos

Para utilizar la aplicación se requiere:

- Dispositivo Android
- Android 8.0 o superior
- Cámara funcional
- Espacio disponible de almacenamiento
- Conexión a internet únicamente para instalación o actualizaciones

---

## 🛠️ Instalación desde Android Studio

### 1. Clonar el repositorio

```bash
git clone https://github.com/USUARIO/Consentir-canitas.git
```

## 2. Abrir el proyecto

1. Abrir Android Studio.
2. Seleccionar Open Project.
3. Elegir la carpeta del proyecto.

## 3. Sincronizar dependencias

Esperar a que Android Studio descargue automáticamente las dependencias de Gradle.

---

## 4. Configurar el dispositivo Android

Para ejecutar la aplicación en un celular Android se deben habilitar las opciones de desarrollador:

1. Ingresar a **Configuración** del celular.
2. Ir a **Acerca del teléfono**.
3. Presionar varias veces sobre **Número de compilación** hasta activar el modo desarrollador.
4. Regresar al menú principal de configuración.
5. Entrar en **Opciones de desarrollador**.
6. Activar la opción **Depuración USB**.

---

## 5. Conectar el dispositivo

1. Conectar el celular al computador mediante cable USB.
2. Aceptar el mensaje de permisos de depuración USB en el dispositivo móvil.
3. Verificar que Android Studio reconozca el dispositivo conectado.

El nombre del celular aparecerá en la parte superior de Android Studio como dispositivo disponible para ejecutar la aplicación.

---

## 6. Ejecutar la aplicación

1. Seleccionar el dispositivo Android conectado.
2. Presionar el botón ▶ **Run App** en Android Studio.
3. Esperar a que Android Studio compile el proyecto e instale la aplicación automáticamente.

Una vez finalizado el proceso, la aplicación quedará instalada y lista para utilizarse en el dispositivo móvil.

---

## 7. Verificar funcionamiento

Después de instalar la aplicación se recomienda realizar las siguientes pruebas:

- Abrir correctamente la aplicación.
- Verificar que el lector QR funcione.
- Registrar un participante de prueba.
- Confirmar que la asistencia quede almacenada.
- Validar que los reportes se generen correctamente.

---

## 📦Instalación mediante APK

Si se cuena con el archivo APK:

1. Copiar el archivo APK al dispositivo Android.
2. Abrir el archivo desde el celular.
3. Permitir instalación desde orígenes desconocidos si el sistema lo solicita.
4. Completar la instalación.
